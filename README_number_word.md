# Number to Word Script

## Description
A Python script that takes numbers as input and returns a word beginning with the same letter as each number.

## Mapping
| Number | Starts With | Word |
|--------|------------|------|
| 0 | Z (Zero) | Zebra |
| 1 | O (One) | Onion |
| 2 | T (Two) | Turtle |
| 3 | T (Three) | Thimble |
| 4 | F (Four) | Falcon |
| 5 | F (Five) | Flamingo |
| 6 | S (Six) | Silver |
| 7 | S (Seven) | Sapphire |
| 8 | E (Eight) | Eclipse |
| 9 | N (Nine) | Nautilus |

For multi-digit numbers, each digit is mapped individually.

## Usage
```bash
python3 number_word.py
```
Then enter numbers one per line. Type `q` to quit.

## Example
```
> 42
42 -> Falcon Turtle
> 7
7 -> Sapphire
> 105
105 -> Onion Zebra Flamingo
```
